======= To do list for running ========

1. git clone 
2. create db with name "user_auth" and "utf8mb4_general_ci" collation on phpmyadmin
3. import "user_auth.sql" file that contained after git clone to the "user_auth" database that you created in step 2
4. if you run api, call this route (http://localhost/User_Auth/api/get_user.php) on browser
5. if you run user register with user level project, call this route (http://localhost/User_Auth/index.php) on browser
